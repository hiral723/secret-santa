# 🎅 Secret Santa Exchange

A simple and fun web app to organize Secret Santa gift exchanges effortlessly. Add participants, generate random assignments, and keep everything fair and anonymous.

---

## 🚀 Features

- Random Assignment: Automatically assigns each participant a Secret Santa  
- No Self-Matching: Ensures no one gets themselves  
- Simple UI: Clean and easy-to-use interface  
- Fast Setup: Works instantly with minimal configuration  
- Firebase Ready: Configured for hosting/deployment  

---

## 🎁 How It Works

1. Enter the names of all participants  
2. Click the "Generate" button  
3. Each participant is assigned a Secret Santa  
4. Keep assignments secret and enjoy the exchange 🎄  

---

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3, JavaScript  
- Backend: Node.js  
- Hosting / Config: Firebase  

---

## 📦 Installation & Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   cd YOUR_REPO_NAME

2. Install dependencies:
   npm install

3. Run the server:
   node server.js

4. Open in browser:
   http://localhost:3000
